---
title: "Process Overview"
---

1. Redirect the requester to the official **Data Request Form**.  
2. The **Data Steward** screens the request.  
3. The **Data Access Committee (DAC)** evaluates the request.  
4. Outcome:
   - Approved  
   - Request more information  
   - Denied  
5. The Data Manager communicates the decision and shares data if approved.
